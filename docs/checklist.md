# Checklist « level up » : passer ton site au niveau entreprise

> Objectif : un site qui inspire confiance, attire des clients qualifiés, et convertit.

## 1) Positionnement & message (fondations)

- [ ] Définir une promesse claire en 1 phrase (« ce que tu fais + pour qui + résultat »).
- [ ] Vérifier que le **hero** répond à 3 questions en 5 secondes : Qui ? Quoi ? Pourquoi toi ?
- [ ] Ajouter une proposition de valeur différenciante (méthode, délai, garantie, niche).
- [ ] Uniformiser le ton de marque (vocabulaire, style, tutoiement/vouvoiement).
- [ ] Créer une hiérarchie de messages : principal, secondaire, preuves.

## 2) Identité visuelle premium

- [ ] Créer/valider un mini brand kit : logo, palette, typo, iconographie.
- [ ] Vérifier la cohérence visuelle sur toutes les pages (espacements, couleurs, boutons).
- [ ] Ajouter un système de composants UI homogène (boutons, cartes, sections, badges).
- [ ] Utiliser des visuels professionnels (photos de projets, mockups, captures avant/après).
- [ ] Préparer une version sombre/contraste fort si pertinente.

## 3) Expérience utilisateur (UX)

- [ ] Clarifier la navigation principale (max 5–6 entrées).
- [ ] Mettre un CTA principal visible dans le header (« Réserver un appel », « Demander un devis »).
- [ ] Structurer chaque page avec logique conversion : Problème → Solution → Preuves → Action.
- [ ] Ajouter des ancres rapides sur les pages longues.
- [ ] Réduire les frictions (formulaires courts, options de contact simples).

## 4) Conversion & business

- [ ] Définir 1 objectif principal par page (prise de rendez-vous, demande devis, etc.).
- [ ] Mettre un CTA sticky mobile si nécessaire.
- [ ] Ajouter des preuves de confiance : témoignages, logos clients, réalisations, chiffres.
- [ ] Afficher une offre claire (packs, à partir de, délais, étapes).
- [ ] Prévoir une page FAQ orientée objections (budget, délais, process, maintenance).
- [ ] Installer un suivi des conversions (form submit, click WhatsApp/téléphone, calendly).

## 5) Contenu qui attire (SEO + crédibilité)

- [ ] Créer des pages services ciblées par intention (pas une page générique).
- [ ] Optimiser les titles/meta descriptions de chaque page importante.
- [ ] Structurer les pages avec H1/H2/H3 orientés mots-clés + lisibilité.
- [ ] Ajouter des cas clients détaillés (contexte, actions, résultats mesurables).
- [ ] Lancer une routine éditoriale (1 contenu utile / semaine ou / 2 semaines).
- [ ] Prévoir des lead magnets (checklist PDF, audit gratuit, guide).

## 6) Performance technique (site rapide = site qui vend)

- [ ] Atteindre de bons Core Web Vitals (LCP, INP, CLS).
- [ ] Compresser les images (WebP/AVIF) et définir des dimensions explicites.
- [ ] Lazy-loader les médias non critiques.
- [ ] Nettoyer les scripts tiers inutiles.
- [ ] Vérifier le rendu mobile réel (pas seulement responsive desktop).
- [ ] Mettre en place cache/CDN correctement.

## 7) SEO technique

- [ ] Vérifier `robots.txt` et `sitemap.xml`.
- [ ] Corriger les erreurs d’indexation (Search Console).
- [ ] Ajouter les données structurées utiles (Organization, LocalBusiness, FAQ, Service).
- [ ] S’assurer qu’il n’y a pas de pages orphelines.
- [ ] Vérifier les canonical tags.
- [ ] Optimiser maillage interne entre services, projets, FAQ, blog.

## 8) Confiance & conformité

- [ ] Mettre à jour mentions légales + politique de confidentialité.
- [ ] Vérifier la conformité RGPD (cookies/consentement si tracking marketing).
- [ ] Afficher des informations de contact complètes et visibles.
- [ ] Ajouter une page « méthode » ou « process » pour rassurer.
- [ ] Ajouter garanties / conditions de collaboration claires.

## 9) Accessibilité (pro = inclusif)

- [ ] Vérifier le contraste des couleurs (WCAG AA minimum).
- [ ] Assurer la navigation clavier (focus visible partout).
- [ ] Ajouter des textes alternatifs descriptifs aux images importantes.
- [ ] Vérifier labels/messages d’erreur des formulaires.
- [ ] Tester lecteur d’écran sur les pages clés.

## 10) Analytics & pilotage

- [ ] Installer GA4/Matomo + Search Console + Tag Manager (si utile).
- [ ] Définir des KPI simples : trafic qualifié, taux conversion, coût lead, prise de contact.
- [ ] Créer un dashboard mensuel.
- [ ] Mettre en place tests A/B sur hero, CTA, offre, formulaire.
- [ ] Faire un rituel d’optimisation continue (toutes les 2–4 semaines).

---

## Plan d’action 30 jours (simple et réaliste)

### Semaine 1 — Base stratégique

- [ ] Clarifier message principal + offre.
- [ ] Ajuster hero + CTA + structure de navigation.
- [ ] Poser tracking minimal (conversions).

### Semaine 2 — Crédibilité

- [ ] Ajouter témoignages + projets + FAQ.
- [ ] Finaliser pages légales et rassurance.
- [ ] Uniformiser design système.

### Semaine 3 — Performance & SEO

- [ ] Optimiser images et vitesse.
- [ ] Corriger SEO technique (sitemap, metas, maillage).
- [ ] Ajouter données structurées.

### Semaine 4 — Conversion

- [ ] Revoir formulaires + friction mobile.
- [ ] Tester 2 variantes hero/CTA.
- [ ] Analyser les chiffres et ajuster.

---

## Priorités si tu veux du résultat rapide (Top 5)

1. [ ] Hero ultra clair + CTA fort au-dessus de la ligne de flottaison.
2. [ ] 3 preuves concrètes (témoignages + projets + chiffres).
3. [ ] Formulaire court + prise de contact fluide (mobile d’abord).
4. [ ] Optimisation vitesse mobile.
5. [ ] Tracking conversions pour piloter les décisions.
